### 1. Spring Scheduler
   
Spring Boot에서 @Scheduled 어노테이션을 사용하여 일정한 시간 간격이나 특정 시간에 코드를 실행할 수 있음.

### 2. Quartz Scheduler
   
JAVA 기반의 Job Scheduling 오픈소스 라이브러리로, 일정 시간마다 지정한 횟수만큼 반복해서 특정 작업을 실행할 수 있음.
